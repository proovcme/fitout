# Character assets

The four rigged base characters in this directory are adapted from the
**Ultimate Animated Character Pack** by Quaternius.

- Source: https://quaternius.com/packs/ultimatedanimatedcharacter.html
- License: Creative Commons Zero v1.0 Universal (CC0-1.0)
- Original models: `Worker_Male`, `Worker_Female`, `Suit_Male`, `Suit_Female`
- Changes for FitOut Simulator: converted from embedded glTF to optimized GLB
  with quantized geometry. Solid colors remain native materials so the models
  do not require blob textures and work under the production CSP.

The original pack is released for personal and commercial use without an
attribution requirement. This notice is kept so the asset origin and license
remain auditable inside the project.
